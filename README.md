🏦 ATM Management System

A desktop-based ATM Management System developed using Java Swing, JDBC, and MySQL. The project simulates the basic functionalities of an ATM, allowing users to securely access their accounts and perform banking operations through a graphical user interface.

📌 Features
👤 User Features
Secure Login using Account Number and PIN
Balance Enquiry
Deposit Money
Withdraw Money
Transfer Funds
View Transaction History
Input Validation and Error Handling


🛠️ Admin Features
Create New Account
View All Accounts
Manage Customer Records
Automatic Account Number Generation

🖥️ Technologies Used
Technology	Purpose
Java	Core Programming Language
Java Swing	Graphical User Interface (GUI)
JDBC	Database Connectivity
MySQL	Database Management
IntelliJ IDEA	Development Environment
